Git mergeStyle

Jun 26, 2022

Git 的 mergeStyle 一共有 3 个选项，分别是 merge（默认）, diff3, zdiff3

可以通过如下命令设置： 

1

git config --global merge.conflictStyle zdiff3

也可以手动编辑 ~/.gitconfig 文件添加以下两行

1
2

[merge]
	conflictStyle = zdiff3
